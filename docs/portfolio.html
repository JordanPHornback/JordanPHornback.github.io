My projects

Project 1 - SMSKite

This is a project built in Java using the Twilio Customer Engagement Platform.

Once downloaded and built, this project creates a .jar file that will populate the GUI shown to the left here. Fill in the "To:" section with who you want to text, write your message, and hit send!

Project 2 - Terminal Messaging App

This project is built in Java and runs in your computer's terminal.

Start up the server on your localhost and then connect as many clients as you would like to have in the chat!

Project 3 - Inventory App

This project was our final group project in our 12 week boot camp. We were tasked with created a full stack application capable of managing inventory. The application has a working back end and front end.

This backend allows users to perform all CRUD actions to the store's database. Users can add new items to the inventory, remove items, updated existing items in the inventory, and view all.

Project 4 - Multiverse E-Learning

This project is a mock-up of an e-learning website similar to Udemy.

The platform features user authentication and authorization, role based CRUD operations varying between the users, admins, and course instructors. Users can enroll in courses, instructors can add/remove courses, and admins can do all of the above and control the roles assigned to registered accounts.

Project 5 - InstaFraud

This project has an educational goal, but it also mimics some of the features and functionalities that major social networking apps like Instagram and Twitter offer.

The main goal of this project was to develop a full stack application that can able run accross all environments with the help of Docker. The main features are user authentication and authorization (registration, signin and refresh token), role based CRUD operation on different posts, follow/unfollow other users, view profile and users reaction feature such as like and comment.
